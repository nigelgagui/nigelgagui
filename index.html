About Me

Hello, my name is Nigel Gagui. I am an IT enthusiast who grew up around technology. I am a Front-end Developer with HTML5 and CSS3 knowledge, currently working on Javascript experience. In 2016, I have completed my diploma in Information Technology in Networking which allowed me to work with multiple operating systems from the open source Linux to the well known Microsoft Windows as well as their server counterparts.

Around in 2008, I have had the interest of looking into private server development of games for educational purposes. I wanted to know how code emulates, what software is to be used to run the code, how they link with each other and many other features they may have or require. It took me a while to understand how everything works until I read more about the background of the source code and helpful comments from other coders in the community. During this time, I was determined to pursue and increase my knowledge around Information Technology.
